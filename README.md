# gitbash_solarized
Solarized (dark) Git Bash and VIM

<hr>

This is how I solarize the Git Bash terminal and vim and change the font, prompt and title. 

# Installation
Make changes into the following files/folders:</br>
~/.bashrc</br>
~/.minttyrc</br>
~/.vimrc</br>
~/.dir_colors</br>
~/.vim/colors/solarized.vim</br>
